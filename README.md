# MySQL-py

\#3 task for Python course with Python mysql.connector

### Prerequisites

```
$ pip3 install mysql-connector
```
Create config.ini file with the following data:
```
[mysql]
host = localhost
database = ***your_db_name***
user = root
password =
```

### Installing

```
$ python3
>>> import connector as cnc
>>> cnc.connect()
```
